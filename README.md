# 💬 Real-Time Chat Application - Frontend

A modern real-time chat application built with React that enables users to communicate instantly with secure authentication and live messaging.

## 🚀 Live Demo

🌐 https://chat-app-frontend-c13c.vercel.app

## 📂 Backend Repository

🔗 https://github.com/Anil-KumarSingh/chat-app-backend

## ✨ Features

- User Registration & Login
- JWT Authentication
- Real-Time Messaging
- Online User Status
- Responsive UI
- Secure API Communication
- Persistent Chat History
- Modern User Experience

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- Axios
- Socket.IO Client
- React Router DOM
- Zustand (State Management)

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Anil-KumarSingh/Chat-app-frontend.git


Navigate into the project:

cd Chat-app-frontend

Install dependencies:

npm install

Create a .env file:

VITE_API_URL=http://localhost:5000

Run locally:

npm run dev
